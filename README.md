# Lets-Practice-list-tuple
## Question-1

You are given two distinct lists `l1` and `l2` of size m and n, their items are not consecutive. Write a `func()` function with two parameters, to find the intersection item between them.

```python
l1 = [3, 2, 4, 5, 1] 
l2 = [7, 4, 5, 2]
func(l1, l2)

output:
[2, 4, 5]
```
